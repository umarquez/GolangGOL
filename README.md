# GolangGOL
Golang Conway's GOL implementation using [_engo_](https://engo.io/) engine.

## Prerequisites
### Setup [engo](https://engo.io/) engine
```shell
go get -u engo.io/engo
```

## Setup
```shell
go get github.com/umarquez/GolangGOL
go build github.com/umarquez/GolangGOL
```
## Run
### Windows
```shell
GolangGOL.exe
```

### Linux
```shell
./GolangGOL
```
